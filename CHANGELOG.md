# Changelog

All notable changes to this project will be documented in this file.

The format is inspired by *Keep a Changelog*.

## \[1.0.0\] - 2026-07-12

### Added

-   User registration and authentication using Spring Security.
-   Secure password encryption with BCrypt.
-   User profile management.
-   Trusted connections management.
-   Money transfer functionality.
-   Transaction history ordered by newest first.
-   Spring Data JPA persistence layer.
-   Thymeleaf user interface.
-   Comprehensive unit tests and MVC tests.
-   Spring Security integration tests.
-   JaCoCo code coverage reports.
-   Maven Surefire HTML reports.
-   Project documentation and README.
